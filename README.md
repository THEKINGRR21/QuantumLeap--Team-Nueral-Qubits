# QuantumLeap--Team-Nueral-Qubits
Hello, we are team neural qubits participating in Quantum Leap 2026. this is the repo for QHack'26

This project focuses on epileptic seizure detection using EEG (Electroencephalogram) feature data. EEG signals are high-dimensional and contain complex nonlinear patterns, making classification challenging for traditional machine learning models.

We propose a Trainable Quantum Kernel-Based Quantum Support Vector Machine (QSVM) that introduces learnable parameters into the quantum feature embedding to improve class separability between seizure and non-seizure states.
